## Task status values
- `done` — completed and verified
- `in-progress` — actively being worked
- `not-started` — defined but not yet begun
- `blocked` — cannot proceed; blocking dependency noted

## Task ID format
`PC-NNNN` where NNNN is a zero-padded sequential number. IDs are permanent and never reused.
